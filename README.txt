The DOCC (Direct Observation of Clinical Practice) plugin transforms your Wordpress installation into a form-based web application.

== Description ==

The DOCC (Direct Observation of Clinical Practice) application is a plugin to Wordpress, an open-source Content Management System. Therefore, you can run a white-labeled version of the DOCC application on your premise by first setting up a Wordpress environment according to your security requirements, then installing and customizing the DOCC plugin and associated software. 

== Installation ==

1.	From the “Plugins” menu, click “Add New”, then click “Upload Plugin” to open “Choose File” dialog.
2.	Drag and drop the provided “DOCC.zip” file onto the “Choose File” button, then click “Install Now”.
3.	Once the installation is complete click “Activate Plugin”. You will be redirected to the “DOCC Setup” menu.
4.	Starting from the “DOCC Setup” menu, follow the on-screen instructions to progress through the pages of the automatic setup.
